Diaaru is a luxury lab-grown diamond brand platform.

**Technical Stack:**
- Next.js 14
- TypeScript
- CSS Modules
- Framer Motion / Intersection Observer for animations

**Status:**
- `CertificationSection` refined with process timeline and explicit grid layout.
- `CollectionsSection` refined with shimmer loading states and high-quality imagery.
- Hydration errors resolved via `suppressHydrationWarning` on HTML tag.
- `WhyChooseUs` component implemented with pure CSS animations and Intersection Observer.
- `Navbar` component rebuilt with glass morphism scroll effect, scroll spy, and mobile menu, and all main sections adjusted with `padding-top` to prevent content overlap.
